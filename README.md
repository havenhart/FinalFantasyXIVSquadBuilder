# Final Fantasy XIV Squad Builder
---
Selects the best four squad members to deploy on a mission based on mission attributes provided.
You can get these values from the training board / roster / mission list.

## Technologies Used
Sveltjs3

C#

.NET Core 2.1

.NET Standard 2.0

Web API 2

Sqlite

Dapper

Bootstrap 4
